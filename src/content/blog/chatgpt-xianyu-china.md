## 三、Step 1：在闲鱼上发布第一个商品

### 3.1 选品：从"简历优化"切入

**为什么不直接做论文？**
- 论文风险高，平台可能下架
- 简历优化需求大、单价适中、复购率高（客户升职后还会找你）
- 操作简单，ChatGPT输出质量稳定

### 3.2 商品标题怎么写？

**错误示范：**
> "代写简历"

太普通，没有差异化，客户为什么要选你？

**正确示范（直接复制修改）：**
> 🔥【AI智能优化】简历修改/简历定制/求职简历/应届生简历/转行简历/英文简历

**标题公式：**
```
[表情符号]【核心卖点】关键词1/关键词2/关键词3/长尾词
```

### 3.3 商品文案模板

**复制这个结构，填入你的信息：**

```
【AI智能简历优化】让你的简历脱颖而出，面试邀约率提升300%

💼 服务内容：
✅ 简历诊断：分析现有简历问题
✅ 内容优化：用AI+人工打磨经历描述
✅ 排版美化：专业模板，HR一眼喜欢
✅ 投递建议：针对不同岗位的简历调整策略

## 四、Step 2：用ChatGPT 5分钟完成一个订单

### 4.1 客户下单后，第一步做什么？

**收集信息（复制这个话术）：**

```
亲，感谢下单！为了给你做出最合适的简历，请提供以下信息：

1. 你的原始简历（Word或PDF）
2. 目标岗位/行业（越具体越好）
3. 1-2个你特别想突出的经历或技能
4. 是否有字数/页数要求？

收到后24小时内交付，急单请告诉我~ 😊
```

### 4.2 ChatGPT优化简历的完整流程

**Step 1：让ChatGPT分析原简历**

```
请分析这份简历的问题，从HR的角度给出改进建议：

[粘贴客户简历内容]

目标岗位：[客户提供的岗位]
```

**Step 2：让ChatGPT重写经历描述**

```
请用STAR法则（情境-任务-行动-结果）重写以下工作经历，要求：
1. 突出量化成果（用数字说话）
2. 使用动词开头
3. 贴合目标岗位的关键词
4. 每点不超过2行

原内容：
[粘贴原经历]

目标岗位：[岗位名称]
该岗位JD中的关键词：[从招聘网站复制]
```

**Step 3：生成完整简历**

```
请基于以下信息，生成一份完整的简历：

个人信息：
[姓名、电话、邮箱等]

求职意向：[岗位]

教育背景：
[粘贴]

工作经历（已优化）：
[粘贴Step2的结果]

项目经历：
[粘贴]

技能证书：
[粘贴]

要求：
1. 使用专业简历格式
2. 控制在一页A4纸
3. 突出与目标岗位相关的经历
```

### 4.3 人工精修（10分钟）

ChatGPT的输出可能有这些问题，需要人工检查：
- **数字太夸张**："提升300%"改成"显著提升"
- **语气太AI**：加一些口语化表达
- **格式问题**：确保标点、换行正确
- **客户特殊要求**：比如客户说"不要提到上家公司名字"

## 五、Step 3：把客户引流到微信，做长期复购

### 5.1 为什么一定要引流到微信？

- 闲鱼抽成高（1%），微信0成本
- 微信可以发文件、语音、长期维护关系
- 客户升职、换工作、朋友需要，都会找你
- 可以卖高客单价服务（面试辅导、职业规划）

### 5.2 引流话术（不违规版）**

**交付时说的话：**

```
简历已经优化好了，请查收！📄

如果后续求职过程中需要调整，或者有面试辅导的需求，可以随时找我。

我的微信：[你的微信号]，备注"简历"，方便后续沟通~ 😊
```

**不要直接说"加我微信"，要说"方便后续沟通"。**

### 5.3 微信朋友圈运营

**每天发1-2条，内容类型：**

**类型1：客户好评（建立信任）**
```
又一位同学拿到offer了！🎉

从简历优化到面试辅导，全程陪伴。
薪资比预期还高了20%，太开心了！

需要简历优化的朋友，点击主页链接下单~ 💼
```

**类型2：干货分享（展示专业）**
```
【简历避坑指南】

这3个词千万不要出现在简历里：
❌ "负责" → 改成 "主导/推动/完成"
❌ "参与" → 改成 "独立完成/核心成员"
❌ "学习能力强" → 改成具体技能

你的简历踩坑了吗？免费诊断，评论区见 👇
```

**类型3：限时优惠（促单）**
```
🔥 本月最后3个优惠名额

简历优化原价59，限时39
送面试高频题+答案模板

手慢无，私信"优惠"抢名额 ⚡

## 七、常见问题 FAQ

**Q1：我不会写简历，能接单吗？**
> 能。ChatGPT会写，你只需要学会怎么问问题。本文的prompt模板直接复制用。

**Q2：客户发现我用AI怎么办？**
> 客户要的是结果，不是过程。只要质量达标，没人关心你怎么做的。而且你本来就要人工精修，不是纯AI输出。

**Q3：闲鱼上竞争很大，怎么脱颖而出？**
> 三点：价格适中（不要最低）、文案专业、响应速度快。前期可以比市场价低20%，积累好评后再涨价。

**Q4：需要投入多少时间？**
> 每天2-3小时，可以接3-5单。月入3000-5000是合理的预期。

**Q5：会不会涉及版权问题？**
> 简历优化是咨询服务，不涉及版权。论文辅助要注意，建议只做"辅导"不做"代写"。
