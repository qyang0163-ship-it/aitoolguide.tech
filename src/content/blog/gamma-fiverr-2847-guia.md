## Las 3 Plantillas de Prompts Que Generan el 80% de Mis Ventas

### Plantilla 1: Pitch Decks para Startups (Más Vendido)

```
Crea un pitch deck profesional para [tipo de startup].

Estructura:
1. Portada: Nombre + tagline impactante
2. Problema: Estadísticas + historia
3. Solución: Producto + demo
4. Mercado: TAM/SAM/SOM
5. Modelo de negocio: Ingresos + proyecciones
6. Competencia: Matriz comparativa
7. Equipo: Fotos + backgrounds
8. Roadmap: Hitos + timeline
9. Inversión: Uso de fondos
10. Cierre: CTA + contacto

Estilo: Minimalista, colores corporativos, tipografía moderna
```

**Ejemplo de uso**: Startups buscando inversión seed/Series A
**Precio**: €150-500 por deck
**Mejor para**: Fundadores, consultores de pitch

### Plantilla 2: Presentaciones Corporativas

```
Diseña una presentación corporativa para [empresa].

Requisitos:
- 10-15 diapositivas
- Paleta de colores: [colores de marca]
- Incluir: Portada, Agenda, 3-5 secciones de contenido, Conclusión
- Gráficos: 2-3 visualizaciones de datos
- Estilo: Profesional, limpio, corporativo
- Transiciones: Suaves y elegantes
```

**Ejemplo de uso**: Reportes trimestrales, propuestas comerciales
**Precio**: €80-200 por presentación
**Mejor para**: Ejecutivos, consultores de negocios

### Plantilla 3: Cursos y Webinars

```
Crea una presentación educativa sobre [tema].

Estructura:
1. Hook: Pregunta provocadora o dato sorprendente
2. Agenda: Lo que aprenderán
3. Módulo 1: Concepto base + ejemplo
4. Módulo 2: Aplicación práctica
5. Módulo 3: Caso de estudio
6. Ejercicio: Actividad interactiva
7. Resumen: Puntos clave
8. Recursos: Links + materiales
9. CTA: Próximos pasos

Estilo: Visual, iconos, espacio en blanco generoso
```

**Ejemplo de uso**: Cursos online, webinars de venta
**Precio**: €100-300 por curso
**Mejor para**: Creadores de contenido, coaches
