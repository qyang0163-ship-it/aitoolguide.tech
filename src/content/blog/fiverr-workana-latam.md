## Fase 2: El Sistema de Entrega con IA (Semana 3-4)

### Flujo de Trabajo: Post de Blog en 15 Minutos

**Cliente ordena un post de 1.500 palabras por €120.**

**Tu flujo:**

**Paso 1: Investigación (2 min)**
```
Prompt: "Dame 5 ángulos únicos para un post sobre [tema] que rankee para [palabra clave]. Incluye un titular atractivo para cada uno."
```

**Paso 2: Esquema (3 min)**
```
Prompt: "Crea un esquema detallado para '[titular elegido]'. Incluye H2s, H3s y puntos clave para cada sección. Objetivo: 1.500 palabras."
```

**Paso 3: Primer Borrador (5 min)**
```
Prompt: "Escribe la sección [X] del post. Usa un tono atractivo y profesional. Incluye ejemplos y consejos accionables."
```

**Paso 4: Editar y Optimizar (5 min)**
- Pasar por Grammarly
- Agregar enlaces internos
- Optimizar para palabra clave objetivo
- Agregar meta descripción

**Tiempo total: 15 minutos.**  
**Tu tarifa efectiva: €480/hora.**

### Sobre-Entregar (El Secreto de las 5 Estrellas)

¿El cliente pidió 1 post? Entrega:
- El post
- 3 variaciones de titular para A/B testing
- Meta descripción optimizada para SEO
- 5 snippets para redes sociales para promocionarlo

**Costo para ti: 3 minutos extra.**  
**Impacto en el cliente: Impactado.**

## Impuestos y Legal (España y LATAM)

### Para España

**Autónomo:**
- Registrarse como autónomo después de €1.000/mes consistentes
- IVA: 21% (general)
- IRPF: Retención del 15% (nuevos autónomos: 7% primer 2 años)

**Herramientas:**
- **Holded** — Facturación y contabilidad
- **Quipu** — Para autónomos
- **Xolo** — Gestoría online (€60/mes)

### Para Latinoamérica

**México:**
- Régimen de Incorporación Fiscal (RIF) para empezar
- Facturación electrónica obligatoria

**Argentina:**
- Monotributo para montos bajos
- Considerar cobrar en USD vía Wise/Payoneer

**Colombia:**
- Régimen simplificado hasta cierto monto
- Payoneer o Wise para recibir pagos internacionales

**General:**
- Guarda el **25-30%** para impuestos
- Usa **Wise** o **Payoneer** para recibir de clientes internacionales
- Lleva registro de TODOS los gastos (herramientas, software, equipo)
