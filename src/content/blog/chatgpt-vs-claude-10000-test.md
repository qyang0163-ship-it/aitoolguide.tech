## 3 Prompts That Made Me the Most Money

### Prompt 1: High-Converting Sales Copy (Claude)

```
Write a sales page for [product] targeting [audience].

Use this structure:
1. Hook: Start with a surprising statistic or question
2. Problem: Agitate the pain point
3. Solution: Present the product as the answer
4. Proof: Include 2-3 specific results/case studies
5. Offer: Make the offer irresistible
6. Urgency: Add time-sensitive bonus
7. CTA: Clear call-to-action

Tone: Conversational but authoritative
Length: 800-1200 words
```

**Used for**: Landing pages, sales emails, VSL scripts
**Average project value**: $1,500-3,000

### Prompt 2: Python Automation Script (ChatGPT)

```
Create a Python script that [specific task].

Requirements:
- Input: [describe input format]
- Output: [describe output format]
- Error handling: Include try-except blocks
- Documentation: Add docstrings and comments
- Testing: Include 3 test cases

Use these libraries: [list libraries]
Follow PEP 8 style guidelines
```

**Used for**: Client automation projects
**Average project value**: $800-2,000

### Prompt 3: Research Synthesis (Claude)

```
Analyze the following information about [topic]:

[paste research data]

Provide:
1. Executive summary (3-5 bullet points)
2. Key insights (5-7 findings)
3. Strategic recommendations (3-5 actions)
4. Risk analysis (potential issues)
5. Next steps (prioritized list)

Format: Professional report style
Length: 1,500-2,000 words
```

**Used for**: Market research reports
**Average project value**: $2,000-5,000
