# ChatGPT Prompt工程完全指南

## 什么是Prompt工程？

Prompt工程（Prompt Engineering）是**设计和优化输入指令**，让AI生成更高质量输出的技术。

同样的AI，不同的Prompt，输出质量可能相差10倍。

**例子：**

❌ 普通Prompt：
```
写一篇关于咖啡的文章
```

✅ 优化后的Prompt：
```
你是一位资深咖啡品鉴师和生活方式博主。

请写一篇关于"手冲咖啡入门"的小红书文案：

目标读者：25-35岁都市白领，刚开始接触咖啡
风格：轻松、亲切、有格调
结构：
1. 痛点引入（为什么手冲咖啡值得学）
2. 3个入门技巧（简单可操作）
3. 推荐一款入门器具（性价比高的）
4. 互动引导

要求：
- 标题要有数字和情绪词
- 正文口语化，像朋友聊天
- 适当使用emoji
- 字数：400-600字
```

**结果差异：**
- 普通Prompt：泛泛而谈，没有针对性
- 优化Prompt：结构清晰、风格明确、可直接使用

## Prompt工程的核心原则

### 原则1：角色设定（Role）

让AI扮演一个特定角色，输出会更专业。

**公式：**
```
你是一位[专业身份]，拥有[相关经验]。
```

**例子：**
- "你是一位有10年经验的资深文案写手"
- "你是一位擅长转化的心理学营销专家"
- "你是一位熟悉Z世代语言的年轻博主"

### 原则2：任务明确（Task）

清晰描述你要AI做什么。

**公式：**
```
请[动作]关于[主题]的[内容类型]
```

**例子：**
- "请写一篇关于时间管理的小红书文案"
- "请为这个产品写5个不同风格的标题"
- "请把这个技术文档改写成大白话"

### 原则3：格式规范（Format）

指定输出的结构和格式。

**常用格式指令：**
- "用 bullet points 列出"
- "分3个段落，每段不超过3句话"
- "用表格对比优缺点"
- "输出JSON格式"

### 原则4：风格指定（Tone）

明确你想要的语气和风格。

**风格词汇：**
- 专业、轻松、幽默、严肃、亲切、权威、俏皮
- 像朋友聊天、像专家授课、像销售推销

### 原则5：约束条件（Constraints）

设置限制条件，避免AI跑偏。

**常用约束：**
- "字数限制在500字以内"
- "不要使用专业术语"
- "避免负面词汇"
- "必须包含以下关键词：..."

## Prompt工程进阶技巧

### 技巧1：少样本学习（Few-Shot）

给AI几个例子，让它学习你的要求。

**例子：**
```
请按照以下风格，为"瑜伽垫"写5个小红书标题：

示例1：
产品：防晒霜
标题：☀️夏天晒不黑！这瓶防晒我囤了3瓶

示例2：
产品：蓝牙耳机
标题：🎧降噪绝了！地铁上终于能听清歌词了

示例3：
产品：保温杯
标题：❄️冬天救命神器！12小时还烫嘴的保温杯

现在写"瑜伽垫"的标题：
```

### 技巧2：思维链（Chain-of-Thought）

让AI展示思考过程，输出更逻辑清晰。

**公式：**
```
请一步一步思考，然后给出答案。
```

**例子：**
```
请分析为什么这个文案转化率低，一步一步思考：

文案：我们的产品很好，功能很多，价格也很便宜，欢迎大家购买。

分析步骤：
1. 先找出文案中的问题
2. 分析每个问题对转化的影响
3. 给出修改建议
4. 重写一个优化版本
```

### 技巧3：自我纠正（Self-Correction）

让AI先输出，再自我检查和改进。

**公式：**
```
先[完成任务]，然后检查[标准]，最后优化。
```

**例子：**
```
请写一篇关于"早起好处"的公众号文章。

步骤：
1. 先写第一版
2. 检查：是否有吸引人的开头？是否有具体案例？结尾是否有行动号召？
3. 根据检查结果优化
4. 输出最终版本
```

### 技巧4：分步骤指令（Step-by-Step）

复杂任务拆分成多个步骤。

**例子：**
```
请帮我策划一场线上分享会：

步骤1：确定主题
- 目标人群：想学习AI副业的上班族
- 时长：1小时
- 请给出3个主题选项，并说明理由

步骤2：设计大纲
- 基于选定的主题，设计详细大纲
- 包括：开场、3个核心内容、互动环节、结尾

步骤3：准备宣传文案
- 写一段吸引人的分享会介绍
- 写3条朋友圈宣传文案
```

## 实战Prompt模板库

### 写作类

**小红书文案：**
```
你是一位资深小红书博主，擅长写爆款种草笔记。

请为[产品名称]写一篇小红书文案：

产品卖点：
- [卖点1]
- [卖点2]
- [卖点3]

目标用户：[用户画像]

要求：
- 标题：有数字+情绪词+emoji
- 正文：分3-4段，每段2-3句话
- 风格：真实、亲切、像朋友推荐
- 字数：300-500字
- 必须包含emoji
- 结尾有互动引导
```

**公众号文章：**
```
你是一位资深公众号写手，擅长写10w+爆文。

主题：[主题]
目标受众：[受众描述]
文章类型：[干货/故事/观点/盘点]

要求：
- 标题：吸引人点击，10-15字
- 开头：用钩子抓住注意力（故事/痛点/数据）
- 正文：3-5个小标题，每部分有案例或数据
- 金句：至少3句加粗
- 结尾：总结+行动号召+互动
- 字数：1500-2000字
- 风格：[轻松/专业/幽默]
```

**产品描述：**
```
你是一位电商文案专家，擅长写高转化率的产品描述。

产品：[产品名称]
核心卖点：
- [卖点1]
- [卖点2]
- [卖点3]

目标用户：[用户画像]
使用场景：[场景描述]

要求：
- 标题：10字以内，突出核心卖点
- 痛点：描述用户遇到的问题
- 解决方案：产品如何解决
- 卖点：用bullet points列出3-5个
- 信任：加入社会证明或保障
- 行动号召：明确的购买引导
- 字数：200-300字
```

### 营销类

**广告文案：**
```
你是一位资深广告文案，擅长写高转化率的广告文案。

产品：[产品名称]
目标受众：[受众描述]
投放平台：[小红书/抖音/朋友圈]
广告目标：[点击/转化/品牌认知]

要求：
- 用AIDA模型（注意-兴趣-欲望-行动）
- 标题：抓人眼球，不超过15字
- 正文：突出核心卖点，解决痛点
- CTA：明确的行动号召
- 字数：[平台要求]
```

**邮件营销：**
```
你是一位邮件营销专家，擅长写高打开率和高点击率的邮件。

邮件类型：[欢迎邮件/促销邮件/召回邮件]
目标：[目标描述]

要求：
- 主题行：吸引人打开，不超过50字
- 开头：个性化称呼，建立连接
- 正文：价值导向，不是硬推销
- CTA：1个明确的行动按钮
- 结尾：友好签名
- 字数：150-300字
```

### 创意类

**品牌命名：**
```
你是一位品牌命名专家，擅长起有记忆点的好名字。

产品类型：[产品类型]
目标用户：[用户画像]
品牌调性：[高端/年轻/专业/有趣]

要求：
- 生成10个品牌名称
- 每个名字说明含义和记忆点
- 检查：是否易读、易记、无负面谐音
- 首选.com域名可用的
```

**Slogan创作：**
```
你是一位Slogan创作专家，擅长写朗朗上口的品牌口号。

品牌：[品牌名称]
产品：[产品描述]
核心卖点：[卖点]
目标用户：[用户画像]

要求：
- 生成10条Slogan
- 每条不超过10个字
- 要朗朗上口，容易记忆
- 体现品牌核心价值
```

## Prompt优化流程

**Step 1：明确目标**
- 我要AI做什么？
- 输出用途是什么？
- 成功标准是什么？

**Step 2：初版Prompt**
- 按角色+任务+格式+风格+约束写第一版

**Step 3：测试输出**
- 运行Prompt，看结果
- 记录问题：太长？太泛？风格不对？

**Step 4：迭代优化**
- 针对性修改：加约束、调风格、给例子
- 再次测试

**Step 5：固化模板**
- 效果好的Prompt保存为模板
- 标注适用场景

## 常见错误与修正

| 错误 | 问题 | 修正 |
|------|------|------|
| 太笼统 | 输出太泛 | 加具体约束和例子 |
| 太复杂 | AI理解不了 | 拆分成多个简单Prompt |
| 太简单 | 输出质量低 | 加角色设定和详细要求 |
| 无约束 | 输出格式乱 | 指定格式和结构 |
| 无例子 | 风格不对 | 给少样本示例 |
