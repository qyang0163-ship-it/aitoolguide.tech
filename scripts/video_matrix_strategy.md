## 视频制作SOP

### 工具栈
1. **脚本生成**：Claude/GPT-4 批量生成脚本变体
2. **视频生成**：
   - Runway Gen-2（AI视频生成）
   - Pika Labs（动画效果）
   - HeyGen（数字人讲解）
3. **剪辑**：CapCut / Premiere Pro
4. **字幕**：剪映自动识别 + 人工校对

### 制作流程（单条15分钟）
```
1. 选择脚本模板 (1min)
2. AI生成画面素材 (5min)
3. 剪辑+配乐+字幕 (7min)
4. 导出+写标题描述 (2min)
```
